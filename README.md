# AI LinkedIn Bio Generator

## Overview

The AI LinkedIn Bio Generator is a web application designed to help users create professional and engaging LinkedIn bios using advanced AI technology. This tool simplifies the process of writing a compelling bio, allowing users to focus on their career achievements and aspirations.

## Features

- **AI-Powered Bio Generation**: Utilizes machine learning algorithms to generate personalized LinkedIn bios based on user input.
- **User-Friendly Interface**: Intuitive design that makes it easy for users to navigate and generate bios quickly.
- **Customization Options**: Users can customize their bios by selecting different styles and tones to match their professional persona.
- **Responsive Design**: Fully responsive layout that works seamlessly on both desktop and mobile devices.

## Technologies Used

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express
- **AI Model**: OpenAI API (or any other AI model used)

## Installation

To get started with the AI LinkedIn Bio Generator, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/mohamedtarekdev/ai-linkedin-bio-generator.git
   cd ai-linkedin-bio-generator
   ```

2. Navigate to the client directory and install dependencies:

   ```bash
   cd client
   npm install
   ```

3. Navigate to the server directory and install dependencies:

   ```bash
   cd ../server
   npm install
   ```

4. Start the server:

   ```bash
   node index.js
   ```

5. Start the client:
   ```bash
   cd ../client
   npm run dev
   ```

## Usage

1. Open your browser and navigate to `http://localhost:5173` (or the port specified).
2. Enter your details in the provided fields.
3. Click on the "Generate Bio" button to create your LinkedIn bio.
4. Review and customize the generated bio as needed.
